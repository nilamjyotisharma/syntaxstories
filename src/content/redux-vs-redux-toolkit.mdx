# Redux vs Redux Toolkit

## Redux

Redux is a predictable state container for JavaScript apps. It helps you write applications that behave consistently, run in different environments (client, server, and native), and are easy to test.

## Redux Toolkit

Redux Toolkit is our official, opinionated, batteries-included toolset for efficient Redux development. It is intended to be the standard way to write Redux logic, and we strongly recommend that you use it.

## Redux Toolkit vs Redux

Redux Toolkit is a package that contains the official, opinionated, batteries-included toolset for efficient Redux development. It is intended to be the standard way to write Redux logic, and we strongly recommend that you use it.

Redux Toolkit is mainly a package that contains other packages that we think are essential for building a Redux app. These include packages like Redux Thunk and Redux DevTools Extension, as well as our own packages like configureStore.
